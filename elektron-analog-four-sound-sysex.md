# Elektron A4 Sound SysEx

OS Version: OS 1.40A

**INFO:** Apparently the patch size has change from 415 to 413 Bytes. Byte 2 + 3 have been removed.


#### Meta

* Header -> 0-21 -> 22 Bytes
* Name -> 22-38 -> 17 Bytes (24 = 0x00; 32 = 0x00)
